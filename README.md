# Filmorate (Rest API)
## Описание
Аналог кинопоиска. Возможности:

- Оценки фильмов;
- Получение популярных фильмов
- Искать фильм;
- Добавлять пользователей в друзья;
- Искать пользователя.

## Технологии
- Java 11; 
- Lombok; 
- Spring Boot; 
- SQL; 
- Maven; 
- Junit; 
- Postman тесты.
